# Whisper Transcriber

Lightweight desktop transcription tool built around Faster Whisper (faster-whisper).
It provides a small popup UI for real-time transcription, a background hotkey daemon,
and helpers to copy results to the clipboard.

## Key features

- Real-time speech-to-text with GPU/CPU fallback
- Frameless popup UI (Tkinter)
- Global hotkey daemon to open or focus the popup
- Clipboard integration and simple installer scripts

## Quick start

1. Install dependencies: `pip install -r requirements.txt`
2. Run the installer: `./install.sh`
3. Start the popup: `./run_popup.sh`
